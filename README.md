# CTF Web Serveur - Timestamp et 32bits

## Présentation du CTF 
**ID** 9 dans **les CTFs de Cyrhades**


Récupérer le flag en réussissant à soumettre un dossier en le faisant passer pour un dossier d'une date comprise entre 1970 à 2000.
